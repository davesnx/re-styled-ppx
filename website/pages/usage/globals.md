### styled.global
**Inject global css**, method to apply general styles to your website.

```reason
[%styled.global {|
  html, body {
    margin: 0;
    padding: 0;
  }
|}];
```

```reason
[%styled.global {|
  html, body {
    margin: 0;
    padding: 0;
  }

  .div {
    display: flex;
  }
|}];
```
